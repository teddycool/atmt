# Illustration Format

## Default Style
- Clean, minimal line art or block diagrams.
- Neutral background (white or very light grey).
- Label only what is essential for understanding — avoid dense text on images.
- Use arrows to show data/control flow direction.
- Colour sparingly: one accent colour per logical group.

## Architecture Diagrams
- Show components as labelled boxes.
- Show MQTT topics as labelled arrows between publisher and subscriber.
- Include field boundary and sensor rays when illustrating truck navigation.

## Notes
- Current image models render dense text poorly — keep on-image labels short.
- Prefer SVG-style descriptions when generating prompts for vector tools.
