# Tools

## Available
| Tool | Notes |
|------|-------|
| Python 3.13 | Runtime for all MapClient scripts |
| paho-mqtt | MQTT client — `pip install paho-mqtt` |
| matplotlib | Visualization — `pip install matplotlib` |
| numpy | Numerics — `pip install numpy` |
| python3-tk | TkAgg backend for matplotlib — `sudo apt install python3-tk` |
| mosquitto_pub / mosquitto_sub | CLI MQTT tools for broker testing |

## Runtime Environment
- Virtual environment at `MapClient/venv/`
- Activate: `source venv/bin/activate`

## Not Available
- MCP server / shared-resources — not configured for this project.
- ROS / any robotics middleware.
